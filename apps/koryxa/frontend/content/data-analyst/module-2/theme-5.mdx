# Thème 5 — Capstone collecte : pack reproductible (RAP + lineage + README + data dictionary)

Objectif : livrer un **Collection Pack** réellement reproductible.

Ce que tu dois produire (dans un ZIP) :
- `final_dataset.csv` (dataset final “user-level”)
- `quality_report.json` (avec `passed` + `failed_checks`)
- `data_dictionary.csv` (documentation des variables)
- `lineage.md` (sources → transformations → outputs)
- `README.md` (comment reproduire)
- `manifest.json` (empreintes SHA256 + environnement)
- scripts (preuve + reproductibilité) :
  - `extract_sql.sql`
  - `extract_api.py`
  - `powerquery.m`
  - `packager.ipynb`

Le notebook packager standardise la structure (dossier `capstone/`), reconstruit `enrolled_at` proprement, génère les preuves (quality/lineage/readme/manifest), puis zippe **toute l’arborescence**.
